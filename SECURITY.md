# Segurança

Não publique segredos, tokens, API keys, arquivos `.env`, logs com credenciais ou dados sensíveis neste repositório.

Se encontrar uma falha de segurança nos prompts/agentes, abra uma issue descrevendo o risco sem expor segredos reais.

Este projeto é um kit de instruções e agentes. Antes de usar em produção, revise permissões, comandos e regras de confirmação conforme seu ambiente.
